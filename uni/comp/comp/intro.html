Compilers: Introduction[..] [top]

A compiler translates a program from a source language into a semantically equivalent program in a target language.

Typically, it translates from languages of higher level to those of lower level of abstraction.

Interpreters, on the other hand, directly execute instructions written in the source language.

Stages of compilation[up]

Or at least those for the compiler we study in this course, designed for the PicoPascal language and Keiko virtual machine.

Lexer

Source character stream → token stream

Parser

Token stream → ast

Type checker

Annotates tree (for codegen's use), does semantic analysis

Codegen

Annotated tree → postfix code

Peephole optmizer

Postfix code → Optimized postfix code

Intermediate codegen

Optimized postfix → operator tree

Optimizer

Further optimization

Instruction selection

Tree → tiled tree, code templates

Register assignment

Code templates → assembly

Assembler

Assembly → machine code
